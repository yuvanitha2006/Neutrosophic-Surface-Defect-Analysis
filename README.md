# Neutrosophic-Surface-Defect-Analysis
Surface defect detection using Neutrosophic Logic

A Flask-based application for detecting industrial surface defects using Neutrosophic Logic.

## Features
- Surface defect detection
- Neutrosophic feature extraction
- CSV feature generation
- Web-based visualization

## Technologies
- Python
- Flask
- OpenCV
- NumPy
- Matplotlib

## Dataset
Due to GitHub size restrictions, the dataset is not included.

Please download it separately and place it inside the `data/` folder.

## How to Run
```bash
pip install -r requirements.txt
python app.py
