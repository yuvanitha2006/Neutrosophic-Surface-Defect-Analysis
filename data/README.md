# Dataset Information

The dataset used in this project is large and cannot be uploaded to GitHub due to file size limitations.

## Dataset Description
The dataset contains industrial surface images used for detecting defects such as cracks, scratches, and dents using Neutrosophic Logic–based analysis.

## Dataset Source
- Source: ( Kaggle )
- Download Link: <PASTE https://www.kaggle.com/datasets/yidazhang07/bridge-cracks-image>
